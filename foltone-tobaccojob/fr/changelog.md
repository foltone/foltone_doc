# Changelog

Toutes les modifications notables de ce script sont documentees dans ce fichier.

## [1.0.0] - Release initiale

### Ajouts
- Pipeline complet de production : recolte de feuilles de tabac, transformation en cigarettes, vente.
- Menu boss RageUI avec gestion des employes, salaires et tenues.
- Coffre de societe partage entre les employes.
- Vestiaire avec tenues par grade.
- Garage de societe avec vehicules par grade.
- Blip configurable sur la carte.
- Systeme de features toggle pour activer/desactiver chaque fonctionnalite.
- Support multilingue (fichiers de traduction modifiables).
- Protection escrow via Tebex.
- Architecture identique aux scripts vigneron/brasseur pour coherence.
