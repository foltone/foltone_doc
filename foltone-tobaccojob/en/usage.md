# Usage

## Overview

The tobacco farming job allows players to harvest tobacco leaves, process them into cigarettes, and sell the finished products. The script includes a full boss menu for society management.

## Work pipeline

### 1. Harvesting

- Go to the harvest point (marked on the map if blip is enabled).
- Approach the harvest zone and interact with the marker.
- A harvesting animation will play. When it finishes, you receive **tobacco leaves** in your inventory.
- Repeat as many times as needed.

### 2. Processing

- Go to the processing point.
- Interact with the marker to open the processing menu.
- Select the desired recipe (e.g., 3 tobacco leaves = 1 cigarette pack).
- Wait for the crafting process to complete.

### 3. Selling

- Go to the sell point.
- Interact with the marker to open the sell menu.
- Select the products and quantity to sell.
- Money is deposited into the society account.

## Boss menu (RageUI)

Accessible only to the boss grade, the menu provides:

### Employee management
- **Hire**: the target player must be nearby.
- **Fire**: select an employee from the list.
- **Promote/Demote**: change an employee's grade.

### Salary management
- Adjust the salary for each grade from the menu.

### Outfit management
- Set the outfit associated with each grade.
- Employees can change at the locker.

### Society storage
- Deposit and withdraw items shared between employees.

### Bank account
- View the society balance.
- Deposit or withdraw money.

## Locker

- Go to the locker location.
- Interact to put on your grade's outfit.
- Interact again to change back to civilian clothes.

## Garage

- Go to the garage point.
- Select a vehicle available for your grade.
- The vehicle spawns at the defined spawn point.
- Return the vehicle to the same point to store it.

## Tips

- Make sure you have enough inventory space before harvesting.
- Sell prices are configured by the server administrator.
- Only the boss can access the management menu. Contact your employer for requests.
