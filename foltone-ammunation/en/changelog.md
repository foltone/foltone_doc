# Changelog

## v1.0.0 - Initial release

### Added
- Ammu-Nation shop system with RageUI menu
- Configurable aisles with weapons, ammo and accessories
- Weapon license system (esx_license integration)
- Store robbery with progress bar
- Police alert with suspect photo and geolocation
- Robbery cancellation if player moves away or NPC dies
- Configurable cooldown between robberies
- Minimum police count requirement for robbery
- Configurable map blips
- Multi-language support (fr, en, es)
