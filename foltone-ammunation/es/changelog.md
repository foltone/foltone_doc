# Changelog

## v1.0.0 - Version inicial

### Agregado
- Sistema de tienda Ammu-Nation con menu RageUI
- Pasillos configurables con armas, municiones y accesorios
- Sistema de licencia de armas (integracion esx_license)
- Robo de tienda con barra de progreso
- Alerta policial con foto del sospechoso y geolocalizacion
- Cancelacion del robo si el jugador se aleja o el NPC muere
- Tiempo de espera configurable entre robos
- Requisito minimo de policias para robar
- Blips configurables en el mapa
- Soporte multilenguaje (fr, en, es)
