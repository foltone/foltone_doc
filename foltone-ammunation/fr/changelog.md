# Changelog

## v1.0.0 - Version initiale

### Ajouts
- Systeme de boutique Ammu-Nation avec menu RageUI
- Rayons configurables avec armes, munitions et accessoires
- Systeme de licence d'arme (integration esx_license)
- Braquage de magasin avec barre de progression
- Alerte police avec photo du suspect et geolocalisation
- Annulation du braquage si le joueur s'eloigne ou si le PNJ meurt
- Cooldown configurable entre les braquages
- Nombre minimum de policiers requis pour braquer
- Blips configurables sur la carte
- Support multilingue (fr, en, es)
