# Changelog

## v1.0.0 - Version inicial

### Agregado
- Sistema de bolsas con menu RageUI
- Multiples tipos de bolsas configurables (pequena, mediana, grande)
- Colocacion en el suelo con modelo 3D
- Almacenamiento de objetos y armas dentro de las bolsas
- Caida de bolsas al morir el jugador
- Opcion de doblar la bolsa para guardar en inventario
- Peso y capacidad configurables por tipo de bolsa
- Limite de bolsas por jugador
- Soporte multilenguaje (fr, en, es)
