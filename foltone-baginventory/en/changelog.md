# Changelog

## v1.0.0 - Initial release

### Added
- Bag system with RageUI menu
- Multiple configurable bag types (small, medium, large)
- Ground placement with 3D model
- Item and weapon storage inside bags
- Bag drop on player death
- Fold bag option to store in inventory
- Configurable weight and capacity per bag type
- Per-player bag limit
- Multi-language support (fr, en, es)
