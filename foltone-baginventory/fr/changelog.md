# Changelog

## v1.0.0 - Version initiale

### Ajouts
- Systeme de sacs avec menu RageUI
- Plusieurs types de sacs configurables (petit, moyen, grand)
- Placement de sacs au sol avec modele 3D
- Stockage d'objets et d'armes dans les sacs
- Lacher de sacs a la mort du joueur
- Option de pliage du sac pour le ranger dans l'inventaire
- Poids et capacite configurables par type de sac
- Limite du nombre de sacs par joueur
- Support multilingue (fr, en, es)
