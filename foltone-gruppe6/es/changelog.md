# Changelog - foltone_gruppe6

## v1.0.0 - Version inicial

### Agregado
- Sistema de empleo Gruppe6 completo con inicio de servicio y misiones
- Sistema de robo de furgoneta blindada con explosivos
- Menu RageUI para gestion del servicio y misiones
- Uniformes configurables por genero
- Recompensas de mision configurables (min/max)
- Requisito minimo de policia para el robo
- Sistema de tiempo de espera entre robos
- Alertas policiales automaticas durante el robo
- Soporte multi-idioma via archivos de traduccion
- Compatible con es_extended y oxmysql
