# Changelog - foltone_gruppe6

## v1.0.0 - Initial Release

### Added
- Complete Gruppe6 job system with clock-in and missions
- Armored van robbery system with explosives
- RageUI menu for service management and missions
- Configurable outfits per gender
- Configurable mission rewards (min/max)
- Minimum police requirement for robbery
- Cooldown system between robberies
- Automatic police alerts during robbery
- Multi-language support via translation files
- Compatible with es_extended and oxmysql
