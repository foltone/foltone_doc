# Changelog - foltone_territories

## v1.0.0 - Version inicial

### Agregado
- Sistema completo de territorios de pandilla
- Captura de zonas mediante venta de drogas a PNJ
- Reacciones de PNJ: aceptar, rechazar, llamar a la policia
- Mejores precios de venta en territorios capturados
- Zonas configurables (posicion, dimensiones, rotacion, puntos requeridos)
- Drogas configurables (precio, puntos de captura, tasas de aceptacion/rechazo)
- Lista de PNJ excluidos (lista negra)
- Trabajos de policia configurables para alertas
- Trabajos de pandilla configurables
- Progresion de captura guardada en base de datos
- Menu RageUI para ventas
- Soporte multi-idioma via archivos de traduccion
- Compatible con es_extended y oxmysql
