# Changelog - foltone_shoprobbery

## v1.0.0 - Initial Release

### Added
- Integrated shop system with item purchase menu
- Complete convenience store robbery system
- Police notification with suspect photo and geolocation
- Automatic cancellation if the player moves or the NPC dies
- Configurable cooldown between robberies per shop
- Configurable minimum police requirement
- Configurable shop items (name, price)
- Configurable shop positions
- Customizable police notification function
- RageUI menu for shop and robbery
- Multi-language support via translation files
- Compatible with es_extended and oxmysql
