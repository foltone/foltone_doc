# Changelog - foltone_shoprobbery

## v1.0.0 - Version inicial

### Agregado
- Sistema de tienda integrado con menu de compra de articulos
- Sistema completo de robo de tienda de conveniencia
- Notificacion policial con foto del sospechoso y geolocalizacion
- Cancelacion automatica si el jugador se mueve o el PNJ muere
- Tiempo de espera configurable entre robos por tienda
- Requisito minimo de policia configurable
- Articulos de tienda configurables (nombre, precio)
- Posiciones de tiendas configurables
- Funcion de notificacion policial personalizable
- Menu RageUI para tienda y robo
- Soporte multi-idioma via archivos de traduccion
- Compatible con es_extended y oxmysql
