# Uso — foltone_brasseur

Mismo flujo que Vigneron: cosechar lúpulo → transformar en cerveza rubia/roja → vender.

Todas las funciones estándar (jefe, cofre, vestuario, garaje, menú F6, duty) funcionan de manera idéntica.
