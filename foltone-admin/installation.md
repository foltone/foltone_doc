# Installation — foltone_admin

## Téléchargement

```bash
cd resources
git clone https://github.com/foltone/foltone_admin
```

## Configuration serveur

```ini
ensure foltone_admin
```

## Dépendances

- **Framework** : ESX Legacy, QB-Core ou Standalone
- **Menu** : RageUI (inclus)

## Commande

Tapez `/admin` en jeu pour ouvrir le menu.
