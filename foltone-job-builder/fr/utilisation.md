# Utilisation — foltone_job_builder

## NUI Builder (carte interactive)

1. Tapez `/jbnui` pour ouvrir l'interface
2. **Sidebar gauche** : liste des jobs existants
3. **Centre** : carte GTA V avec marqueurs colores
4. **Panel droit** : editeur a onglets (General, Grades, Points, Coffres, Garage, Processing, Annonces)

### Placer des points
- Cliquez sur "Placer" a cote d'un point
- Cliquez sur la carte pour positionner
- Ou cliquez "Ma position" pour utiliser votre position in-game

### Pipeline de processing
Ajoutez des etapes de recolte/transformation/vente avec items, durees, animations et prix.

### Multi-coffres
Ajoutez plusieurs coffres avec grade minimum requis.

## Duplication

Cliquez sur "Dupliquer" pour copier un job existant avec un nouveau nom.

## RageUI Builder

Le builder RageUI (`/jb`) offre les memes fonctionnalites via des menus classiques.

## Securite

Seuls les admins (`Config.AdminGroups`) peuvent creer/modifier/supprimer des jobs. Tous les events serveur sont securises avec rate limiting.
