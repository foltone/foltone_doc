# Usage — foltone_job_builder

## NUI Builder (`/jbnui`)
Interactive GTA V map. Click to place points. Tabs: General, Grades, Points, Storages, Garage, Processing, Ads. Duplicate jobs in one click.

## RageUI Builder (`/jb`)
Same features via classic RageUI menus.

## Security
Admin-only access. All server events rate-limited and validated.
