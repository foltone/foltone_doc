# Configuracion — foltone_advanced_ambulancejob

## Sistema de muerte

- Timer antes de boton "llamar EMS" (5 min por defecto)
- Timer antes de respawn automatico (10 min)
- Penalizaciones configurables al respawn: items, dinero, armas

## Camillas

3 tipos con 3 posiciones cada uno (arriba, abajo, sentado). Animaciones personalizadas.

## Farmacia

Lista de items medicos con limites configurables.
