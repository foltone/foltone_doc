# Usage — foltone_advanced_ambulancejob

## Death system (NUI)

Player dies > NUI timer > EMS call button > EMS revive or auto-respawn with penalties.

## F6 Menu

Heal, big heal, revive, stretchers, escort, props, pharmacy.

## Admin commands

- `/revive [id]` — Revive a player
- `/reviveall` — Revive all players

## Standard features

Boss, storage, locker, garage, duty — same as all Foltone v2.0 jobs.
