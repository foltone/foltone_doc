# Configuration — foltone_advanced_ambulancejob

## Death system

- `FirstTimer`: time before "call EMS" button (default 5 min)
- `SecondTimer`: time before auto-respawn (default 10 min)
- Configurable penalties on respawn: clear items, money, weapons

## Medical care

Configurable animation durations for heal, big heal, revive.

## Stretchers

3 types (combicarrier, fernocot, strykergurney) with 3 positions each (up, down, sit). Custom animations.

## Pharmacy

Configurable item list with per-item max limits.

## Heli garage

Separate helicopter garage on hospital roof.
