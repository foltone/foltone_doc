# Changelog

## v1.0.0 - Initial release

### Added
- Road code theory exam with NUI interface
- Configurable questions with multiple choice and timer
- Configurable minimum passing score
- Car practical exam with checkpoint route
- Motorcycle practical exam with dedicated route
- Truck practical exam with reduced error tolerance
- Infraction detection (speeding, collisions, missed checkpoints)
- Automatic license granting via esx_license
- Configurable exam vehicles
- Instructor NPC with map blip
- Multi-language support (fr, en, es)
