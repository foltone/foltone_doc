# Changelog

## v1.0.0 - Version initiale

### Ajouts
- Examen du code de la route avec interface NUI
- Questions configurables avec choix multiples et timer
- Score minimum configurable pour la reussite
- Examen pratique voiture avec parcours a checkpoints
- Examen pratique moto avec parcours dedie
- Examen pratique poids lourd avec tolerance reduite
- Detection des infractions (exces de vitesse, collisions, checkpoints manques)
- Attribution automatique des licences via esx_license
- Vehicules d'examen configurables
- PNJ moniteur avec blip sur la carte
- Support multilingue (fr, en, es)
