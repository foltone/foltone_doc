# Changelog

## v1.0.0 - Version inicial

### Agregado
- Examen teorico de codigo de trafico con interfaz NUI
- Preguntas configurables con opcion multiple y temporizador
- Puntuacion minima configurable para aprobar
- Examen practico de coche con ruta de puntos de control
- Examen practico de moto con ruta dedicada
- Examen practico de camion con tolerancia reducida
- Deteccion de infracciones (exceso de velocidad, colisiones, puntos de control perdidos)
- Otorgamiento automatico de licencias via esx_license
- Vehiculos de examen configurables
- NPC instructor con blip en el mapa
- Soporte multilenguaje (fr, en, es)
