# Usage — foltone_gouvernement

## F6 Menu
Adapts based on branch: admin sees taxes/nominations/permits, security sees handcuffs/escort/armory.

## Tax system
- Paycheck: automatic deduction on ESX payday
- Society cronjob: periodic tax on all target societies
- Invoices: governor sends manual tax bill to society boss

## Nominations
Select job > player > grade. Applies immediately (online) or via DB (offline). Optional broadcast.

## Standard features
Boss, storage, locker, garage, duty — same as all Foltone v2.0 jobs.
