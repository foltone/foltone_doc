# Installation — foltone_gouvernement

Requirements: ESX Legacy, ox_lib, oxmysql, skinchanger, esx_society. Optional: ox_inventory, ox_target, esx_billing (tax invoices), esx_license (permits).

Run `foltone_gouvernement.sql` to create job, 8 grades, accounts and storage.
