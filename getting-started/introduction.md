# Bienvenue

Bienvenue dans la documentation officielle des scripts **Foltone** pour FiveM.

## À propos

Foltone développe des scripts FiveM optimisés, documentés et compatibles avec les principaux frameworks :

- **QB-Core**
- **ESX Legacy**
- **Standalone**

## Comment utiliser cette doc

Chaque script a sa propre section avec :

- **Installation** — Comment installer le script sur votre serveur
- **Configuration** — Les fichiers de config et options disponibles
- **Changelog** — L'historique des versions et mises à jour
