# Changelog

Todos los cambios notables de este script se documentan en este archivo.

## [1.0.0] - Lanzamiento inicial

### Agregado
- Servicio de bar con bebidas configurables (cervezas, vinos, cocteles, licores).
- Sistema de farming de ingredientes en puntos configurables.
- Menu de jefe RageUI con gestion de empleados, salarios y uniformes.
- Almacen de sociedad compartido entre empleados.
- Vestuario con uniformes por grado.
- Garaje de sociedad con vehiculos por grado.
- Blip configurable en el mapa.
- Sistema de activacion de funcionalidades.
- Soporte multilenguaje (archivos de traduccion editables).
- Proteccion escrow via Tebex.
