# Changelog

All notable changes to this script are documented in this file.

## [1.0.0] - Initial release

### Added
- Bar service with configurable drinks (beers, wines, cocktails, spirits).
- Ingredient farming system at configurable points.
- RageUI boss menu with employee, salary, and outfit management.
- Shared society storage for employees.
- Locker with grade-based outfits.
- Society garage with grade-based vehicles.
- Configurable map blip.
- Features toggle system.
- Multi-language support (editable translation files).
- Escrow protection via Tebex.
