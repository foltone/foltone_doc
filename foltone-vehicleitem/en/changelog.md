# Changelog

All notable changes to this script are documented in this file.

## [1.0.0] - Initial release

### Added
- Vehicle purchase system as inventory items.
- Spawn vehicles from inventory.
- Store vehicles at any time (convert back to item).
- Shop with vendor NPC and RageUI menu.
- Fully configurable vehicle list (model, label, item, price).
- Configurable map blip.
- Compatible with standard ESX inventory systems.
- Vehicle item trading between players.
