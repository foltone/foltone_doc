# Changelog

Todos los cambios notables de este script se documentan en este archivo.

## [1.0.0] - Lanzamiento inicial

### Agregado
- Sistema de compra de vehiculos como items de inventario.
- Spawn de vehiculos desde el inventario.
- Guardado de vehiculos en cualquier momento (conversion a item).
- Tienda con NPC vendedor y menu RageUI.
- Lista de vehiculos completamente configurable (modelo, label, item, precio).
- Blip configurable en el mapa.
- Compatible con los sistemas de inventario estandar de ESX.
- Intercambio de items de vehiculos entre jugadores.
