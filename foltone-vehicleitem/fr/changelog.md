# Changelog

Toutes les modifications notables de ce script sont documentees dans ce fichier.

## [1.0.0] - Release initiale

### Ajouts
- Systeme d'achat de vehicules sous forme d'items dans l'inventaire.
- Spawn de vehicules depuis l'inventaire.
- Rangement de vehicules a tout moment (retour en item).
- Boutique avec PNJ vendeur et menu RageUI.
- Liste de vehicules entierement configurable (modele, label, item, prix).
- Blip configurable sur la carte.
- Compatible avec les systemes d'inventaire ESX standard.
- Echange d'items vehicules entre joueurs.
