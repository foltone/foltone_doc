# Changelog - foltone_lscustoms

## v1.0.0 - Initial Release

### Added
- Standalone vehicle customization menu (no job required)
- Live preview for all modifications
- Aesthetic modifications: spoiler, bumpers, skirts, exhaust, hood, roof, wheels, horn, windows, plates, xenon
- Performance upgrades: engine, brakes, transmission, suspension, turbo
- Paint system: primary, secondary, pearlescent, wheel color
- Vehicle repair and cleaning
- NPC towing missions with rewards
- Player-to-player invoice billing system
- RageUI menu with intuitive navigation
- Compatible with es_extended and oxmysql
