# Changelog - foltone_lscustoms

## v1.0.0 - Version inicial

### Agregado
- Menu standalone de personalizacion de vehiculo (sin empleo requerido)
- Vista previa en vivo para todas las modificaciones
- Modificaciones esteticas: spoiler, parachoques, faldones, escape, capo, techo, llantas, bocina, ventanas, placas, xenon
- Mejoras de rendimiento: motor, frenos, transmision, suspension, turbo
- Sistema de pintura: primaria, secundaria, nacarada, color de llantas
- Reparacion y limpieza de vehiculo
- Misiones de remolque NPC con recompensas
- Sistema de facturacion entre jugadores
- Menu RageUI con navegacion intuitiva
- Compatible con es_extended y oxmysql
