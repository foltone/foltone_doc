# Configuration — foltone_policejob

## Police-specific features

- **K9**: configurable dog model
- **Props**: cones, barriers, gazebo (placeable/removable)
- **Armory**: weapons/items by grade
- **Jail**: cells + prison with NUI timer, prisoner clothes
- **Fines**: 5 categories with configurable amounts
- **Radio codes**: calls + help requests with map blips
- **Heli garage**: separate helicopter garage
