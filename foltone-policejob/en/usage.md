# Usage — foltone_policejob

## F6 Menu

Handcuffs, escort, search, fines, jail (custody/prison with NUI timer), K9 (spawn, follow, attack, search), props, radio codes, reinforcement requests, armory, billing.

## Jail system

- **Custody**: teleport to cell with prisoner clothes, timed release
- **Prison**: teleport to prison with NUI countdown timer

## Standard features

Boss, storage, locker, garage, duty — same as all Foltone v2.0 jobs.
