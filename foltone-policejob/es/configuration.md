# Configuracion — foltone_policejob

## Funciones policiales

- **K9**: modelo de perro configurable
- **Props**: conos, barreras, gazebo
- **Armeria**: armas/items por grado
- **Prision**: celdas + prision con timer NUI, ropa de prisionero
- **Multas**: 5 categorias con montos configurables
- **Codigos radio**: llamadas + solicitudes de refuerzo con blips
- **Garaje helicoptero**: garaje separado
