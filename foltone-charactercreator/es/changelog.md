# Changelog

## v1.0.0 - Version inicial

### Agregado
- Creador de personajes completo con menu RageUI
- Personalizacion detallada del rostro (nariz, pomulos, mandibula, menton, ojos, labios)
- Sistema de herencia (mezcla de rasgos de padres)
- Seleccion de peinados con colores y mechas
- Barba y vello facial (personajes masculinos)
- Personalizacion de ropa (parte superior, pantalones, zapatos)
- Accesorios (sombreros, gafas, aretes, cadenas, relojes)
- Vista previa en tiempo real de los cambios
- Integracion con skinchanger para guardar
- Soporte multilenguaje (fr, en, es)
