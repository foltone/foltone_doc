# Usage

## Character creation

The character creator appears automatically on a player's first connection.

### Creation steps

#### 1. Gender

Choose between a male or female character. This choice determines available clothing models and hairstyles.

#### 2. Heritage (parents)

If enabled in the configuration:

- Select a father and mother model.
- Adjust the resemblance slider to blend traits.
- The face updates in real time.

#### 3. Face shape

Adjust facial features with sliders:

- Nose width and height
- Nose bridge
- Cheekbone width and height
- Jaw width and height
- Chin length
- Eye opening
- Lip thickness

Each slider ranges from -1.0 to 1.0 for precise control.

#### 4. Hair

- Choose a hairstyle from the available models.
- Select a primary color.
- Optionally add highlights in a different color.

#### 5. Beard and facial hair (male characters)

- Select a beard style.
- Adjust opacity.
- Choose the color.

#### 6. Clothing

Customize each clothing piece:

- Top (t-shirt and jacket)
- Pants
- Shoes

#### 7. Accessories

- Hats and caps
- Glasses
- Earrings
- Chains
- Watches

#### 8. Confirmation

Confirm your appearance to save the character. The appearance is stored via skinchanger and can be modified later if a modification system is in place.

## Menu navigation

- **Up/Down**: Navigate between options
- **Left/Right**: Adjust values (sliders, models)
- **Enter**: Confirm selection
- **Backspace**: Return to previous menu

## Later modification

The creator only appears on first connection. To modify appearance later, a plastic surgery or wardrobe script is needed (not included).
