# Changelog

## v1.0.0 - Initial release

### Added
- Full character creator with RageUI menu
- Detailed face customization (nose, cheekbones, jaw, chin, eyes, lips)
- Heritage system (parent trait blending)
- Hairstyle selection with colors and highlights
- Beard and facial hair (male characters)
- Clothing customization (tops, pants, shoes)
- Accessories (hats, glasses, earrings, chains, watches)
- Real-time preview of changes
- Skinchanger integration for saving
- Multi-language support (fr, en, es)
