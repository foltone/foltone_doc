# Installation — Tattoo Shop

## Téléchargement

```bash
cd resources
git clone https://github.com/foltone/foltone_tattooshop
```

## Configuration serveur

```ini
ensure foltone_tattooshop
```

## Dépendances

- **Framework** : ESX Legacy, QB-Core ou Standalone
- **Menu** : RageUI (inclus)

## Fonctionnalités

- Tous les tatouages GTA V disponibles
- Interface RageUI intuitive
- Preview en temps réel
- Sauvegarde persistante
- Prix configurables
