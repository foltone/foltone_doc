# Registro de cambios

## v1.0.0 — 2026-03-16

### Añadido
- Bot de Discord integrado directamente en el recurso de FiveM (Node.js embebido)
- 30 comandos slash de administración con sistema de permisos basado en roles de Discord
- Monitorización en tiempo real: embeds enriquecidos con botones de acción directa
- Dashboard con actualización automática con estadísticas del servidor, lista de jugadores, eventos recientes
- Sistema de tickets bidireccional Discord <-> FiveM con transcripciones automáticas
- Chat sincronizado en tiempo real entre Discord y el chat del juego
- Alertas inteligentes: advertencias excesivas, transacciones sospechosas, desconexión masiva, lista de vigilancia
- Lista de vigilancia: monitorización y alertas de conexión para jugadores marcados
- Programador de tareas: reinicios automáticos con cuenta regresiva, anuncios programados
- Panel NUI de administración completo (6 páginas): Dashboard, Jugadores, Economía, Registros, Tickets, Programador
- Compatible con ESX Legacy, QBCore y QBX (detección automática)
- Sistema de registros multicanal (conexiones, muertes, transacciones, acciones de administradores, chat)
- Retención automática de registros con purga configurable
- Soporte para ACE, vMenu, txAdmin, EasyAdmin y sistemas de administración personalizados
