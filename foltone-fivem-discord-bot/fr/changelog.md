# Changelog

## v1.0.0 — 2026-03-16

### Ajouts
- Bot Discord integre directement dans la resource FiveM (Node.js embarque)
- 30 commandes slash admin avec systeme de permissions par role Discord
- Monitoring en temps reel : embeds enrichis avec boutons d'action directe
- Dashboard auto-refresh avec statistiques serveur, liste joueurs, derniers evenements
- Systeme de tickets bidirectionnel Discord ↔ FiveM avec transcripts automatiques
- Chat synchronise en temps reel entre Discord et le chat in-game
- Alertes intelligentes : warns excessifs, transactions suspectes, crash massif, watchlist
- Watchlist : surveillance et alertes a la connexion de joueurs marques
- Taches programmees : redemarrages automatiques avec countdown, annonces planifiees
- Panel NUI admin complet (6 pages) : Dashboard, Joueurs, Economie, Logs, Tickets, Planificateur
- Compatible ESX Legacy, QBCore et QBX (detection automatique)
- Systeme de logs multi-canaux (connexions, morts, transactions, actions admin, chat)
- Retention automatique des logs avec purge configurable
- Support des systemes admin ACE, vMenu, txAdmin, EasyAdmin et custom
