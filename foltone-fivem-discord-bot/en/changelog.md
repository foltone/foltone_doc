# Changelog

## v1.0.0 — 2026-03-16

### Added
- Discord bot integrated directly into the FiveM resource (embedded Node.js)
- 30 admin slash commands with role-based Discord permission system
- Real-time monitoring: rich embeds with direct action buttons
- Auto-refresh dashboard with server stats, player list, recent events
- Bidirectional ticket system Discord <-> FiveM with automatic transcripts
- Real-time synchronized chat between Discord and in-game chat
- Smart alerts: excessive warns, suspicious transactions, mass crash, watchlist
- Watchlist: monitoring and connection alerts for flagged players
- Task scheduler: automatic restarts with countdown, scheduled announcements
- Full admin NUI panel (6 pages): Dashboard, Players, Economy, Logs, Tickets, Scheduler
- Compatible with ESX Legacy, QBCore, and QBX (automatic detection)
- Multi-channel log system (connections, deaths, transactions, admin actions, chat)
- Automatic log retention with configurable purge
- Support for ACE, vMenu, txAdmin, EasyAdmin, and custom admin systems
